Animaciones de carga

Al crear páginas web que requieren de ciertos proceso, e incluso para dar dinamismo a una página, una práctica que muchos utilizan es añadir una animación de carga, en este apartado podrás visualizar distintos diseños que he realizado para que puedas utilizarlos libremente en tus proyectos

Los recursos necesarios para poder utilizarlos puedes descargarlos a continuación:

Primer diseño

Ideal para cuando solo necesitas mostrar algo que le haga saber a tus usuarios que tu página está realizando un proceso de manera minimalista

¿Cómo aplicarlo a mi página web?

Ahora que ya lo has agregado, debes trabajar con javascript, no te preocupes, en los recursos ya está el código que lo permite funcionar, sin embargo debes tener unas cosas en claro

  • Debes tener un contenedor específico para tu contenido, el contenido va a ser siempre el elemento a controlar, este contenedor debe tener un id llamado "content-gral" ya que por ese id el javascript de los recursos puede acceder a el
  • El elemento de carga debe de ir en la parte superior de tu documento HTML para que no te genere confusiones
