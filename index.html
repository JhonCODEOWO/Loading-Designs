Animaciones de carga

Al crear páginas web que requieren de ciertos proceso, e incluso para dar dinamismo a una página, una práctica que muchos utilizan es añadir una animación de carga, en este apartado podrás visualizar distintos diseños que he realizado para que puedas utilizarlos libremente en tus proyectos

Los recursos necesarios para poder utilizarlos puedes descargarlos a continuación:

Primer diseño

Ideal para cuando solo necesitas mostrar algo que le haga saber a tus usuarios que tu página está realizando un proceso de manera minimalista

Segundo diseño

Variable más colorida que permite dar un toque más ánimado

Tercer diseño

Barras de carga que puedes usar para ser más específico con lo que está realizando tu aplicación web
